/api/add/board

Adds a board at a given route.
JSON Body Parameters: {"title" : title_of_board, "route", route_of_board}
JSON Response: {"error" : error_message} (error is blank if there is no error message)

/api/add/thread

Adds a thread to the given board.
JSON Body Parameters: {"board" : board_id_to_post, "title" : title_of_thread, "message", content} (message is content of first post)
JSON Response: {"error" : error_message, "username" : user_name} (error is blank if there is no error message, username is the current logged in user or Anonymous if nobody is logged in)

/api/add/post

Adds a post to the given thread.
JSON Body Parameters: {"thread" : thread_id_to_post, "title" : title_of_post, "message", content}
JSON Response: {"error" : error_message, "username" : user_name} (error is blank if there is no error message, username is the current logged in user or Anonymous if nobody is logged in)

/api/get/boardid

Gets the id of the board corresponding to the given route.
JSON Body Parameters: {"route" : route_of_board}
JSON Response: {"id" : id_of_board}

/api/get/boards

Gets a list of all of the boards.
JSON Response: [{"title" : board_title, "route" : board_route, "numthreads" : board_number_of_threads, "datemod" : board_date_modified}, ...]

/api/get/threads

Gets a list of threads in a given board.
JSON Body Parameters: {"board" : board_id}
JSON Response: [{"id" : thread_id, "title" : thread_title, "board" : thread_board_id, "numposts" : thread_number_of_posts, "datemod" : thread_date_modified}, ...]

/api/get/posts

Gets a list of posts in a given thread.
JSON Body Parameters: {"thread" : thread_id}
JSON Response: [{"id" : post_id, "user" : post_user, "thread" : post_thread_id, "title" : post_title, "message" : post_message, "datemod" : post_date_modified}, ...]
